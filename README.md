# Hi there 👋 I’m Nathee Srina (นที ศรีนะ)

🎓 Lecturer at **Long Industrial and Community Education College** 📚 Teaching in **Electronics** and **Computer Engineering Technology** 💻 **Full-Stack Developer & Maker** passionate about **Open Source, IoT, Local AI, and HomeLab Infrastructure** 🚀 Bridging the gap between **academic theory, software development, and hardware integration**

---

## 👨‍🏫 About Me
- 👨‍💻 **Role:** A Full-stack developer and educator who loves building systems from the ground up—from writing code to configuring servers.
- 🔭 **Currently working on:** Full-stack applications (Next.js, PostgreSQL, Docker) and an IoT (ARDUINO, ESP32, ESP-NOW, RPI).
- 🖥️ **Infrastructure & HomeLab:** Managing virtualized environments and self-hosting services using Proxmox, Docker, and Linux (Ubuntu/Mint, Fedora/CachyOS).
- 🌱 **Currently exploring:** Running and integrating Local AI models (Ollama) into practical applications.
- 👨‍🏫 **Teaching & Research:** Developing hands-on IoT curriculums and researching Computer Vision/Machine Learning (e.g., real-time safety glasses detection).
- ⚡ **Fun fact:** When I'm not configuring servers, coding, or teaching, I'm probably catching up on the classic 1979 Mobile Suit Gundam series!
